MBI - Mass Balance Index (calculation:: http://www.saga-gis.org/saga_module_doc/2.1.3/ta_morphometry_10.html)
LS-factor - slope length (LS) factor as used by the Universal Soil Loss Equation (USLE) (calculation: http://www.saga-gis.org/saga_module_doc/2.1.3/ta_hydrology_22.html)
RSP - Arithmetic combination of VDChN and VDCuN: VDChN / (VDChN + VDCuN)
VDChN - Vertical distance to channel network (calculation: http://www.saga-gis.org/saga_module_doc/2.1.3/ta_channels_3.html)
VDCuN - Vertical distance to culmination network (calculation: same as VDChN applied for an inverted DEM)
